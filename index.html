Current rules

A focused ruleset with practical early wins.

Legacyless is intentionally starting narrow. Each rule is meant to be human-friendly, easy to audit, and strong enough to earn a place in an automated modernization workflow.

Rule 01 nullptr modernization

Replace legacy null pointer constants

Converts NULL or literal 0 to nullptr when they are being used as null pointer constants, improving clarity and matching modern C++ intent.

Before

Socket* socket = NULL;
connect(socket, 0);

After

Socket* socket = nullptr;
connect(socket, nullptr);
See rule details
Rule 02 missing override

Add missing override

Marks derived member functions with override when they are already overriding a virtual base declaration, making intent explicit and strengthening compiler checking.

Before

struct Derived : Base {
  void render();
};

After

struct Derived : Base {
  void render() override;
};
See rule details
